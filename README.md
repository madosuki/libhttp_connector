# libhttp_connector

This program is simple and basic http connector library.  
support Linux and MinGW.  
required OpenSSL.  
using build tools of CMake.


```
mkdir -p build && cd build
```

then  
## Linux  
```
cmake .. && cmake --build .
```
   
## MSYS2 with MinGW
```   
cmake .. -G "MSYS Makefiles" && cmake --build .
```
